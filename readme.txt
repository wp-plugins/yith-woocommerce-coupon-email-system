YITH WooCommerce Coupon Email System offers an automatic way to send a coupon to your users according to specific events.

== Description ==

YITH WooCommerce Coupon Email System let you send an automatic rewarding email for the first purchase of your customers

Rewarding your customer is vital: they will be loyalize and will come back in your shop to purchase more and more products.
YITH WooCommerce Coupon Email System helps in that, creating an automatic process that lets you send a coupon via email with the first purchase they will make in your shop. A powerful tool to get customers' attention and make them spend more in your online shop.

**Features:**

* Enable/disable the plugin from option panel
* Select mail type (HTML or plain text)
* Customize email subject
* Customize email content using a wide set of options
* Send test email

Full documentation is available [here](http://yithemes.com/docs-plugins/yith-woocommerce-coupon-email-system/).


== Installation ==

= Installation =

Important: before of all you have to download and activate WooCommerce plugin, which is mandatory for YITH WooCommerce Coupon Email System to be working. If WooCommerce plugin is active, you just need to activate the YITH WooCommerce Coupon Email System plugin.

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH WooCommerce Coupon Email System` from Plugins page.

= Configuration =

YITH WooCommerce Coupon Email System will add a new tab called "Coupon Email System" inside the YIT Plugins menu item. There you are able to configure all plugin settings.
